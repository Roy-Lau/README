# 				vuepress 开发分支

> 使用 `vuepress` 开发博客


### clone && build && dev


```sh
# 下载
git clone git@github.com:roy-lau/blog.git --b dev blog-dev --depth=1

# 进入目录
cd blog-dev

# 安装依赖
npm i

# 运行项目
# npm run dev

# 构建项目
npm run build
```
